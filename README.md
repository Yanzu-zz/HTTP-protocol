HTTP协议原理 + 实践项目学习
===============

这是一个HTTP练习项目，其中 HTTP_basic 目录记了HTTP协议基础及发展历史，src/*下是 HTTP 功能实践。

### 功能目录架构说明

```
└─src
    ├─accept         数据协商
    ├─cache-control  缓存控制和 etag
    ├─connection     HTTP 长连接
    ├─cookie         cookie
    ├─cors           跨域相关
    ├─csp            Content Security Policy 内容安全策略
    ├─http           创建一个简易 HTTP 服务器
    ├─nginx-cache    nginx-cache 相关
    └─redirect       重定向
```

## 运行
```
    npm install
    npm run server
```
    然后点击你想要查看的页面即可
